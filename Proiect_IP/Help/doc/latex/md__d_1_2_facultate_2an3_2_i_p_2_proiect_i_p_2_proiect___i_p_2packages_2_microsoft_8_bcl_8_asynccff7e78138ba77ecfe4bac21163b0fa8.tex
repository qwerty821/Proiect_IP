 As of C\# 8, the C\# language has support for producing and consuming asynchronous iterators. The library types in support of those features are available in .NET Core 3.\+0 and newer as well as in .NET Standard 2.\+1. This library provides the necessary definitions of those types to support these language features on .NET Framework and on .NET Standard 2.\+0. This library is not necessary nor recommended when targeting versions of .NET that include the relevant support.\hypertarget{md__d_1_2_facultate_2an3_2_i_p_2_proiect_i_p_2_proiect___i_p_2packages_2_microsoft_8_bcl_8_asynccff7e78138ba77ecfe4bac21163b0fa8_autotoc_md1}{}\doxysection{\texorpdfstring{Key Features}{Key Features}}\label{md__d_1_2_facultate_2an3_2_i_p_2_proiect_i_p_2_proiect___i_p_2packages_2_microsoft_8_bcl_8_asynccff7e78138ba77ecfe4bac21163b0fa8_autotoc_md1}
